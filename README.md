# 🛒 PHP E-commerce Store - Full Stack Project

[[PHP](https://img.shields.io/badge/PHP-7.4+-777BB4?style=flat&logo=php)](https://php.net)
[[MySQL](https://img.shields.io/badge/MySQL-5.7+-4479A1?style=flat&logo=mysql)](https://mysql.com)

A complete e-commerce platform built with PHP & MySQL featuring user and admin panels.

## ✨ Key Features
- **User Side:** Account registration, product browsing, shopping cart, secure checkout, order tracking
- **Admin Panel:** Dashboard with statistics, CRUD for products/categories/users, order management
- **Security:** PDO with prepared statements, password_hash(), session validation
- **Database:** Normalized MySQL schema with relational integrity

## ⚙️ Installation
1. Clone the repository
2. Import `shopping.sql` into MySQL
3. Rename `include/connected_example.php` to `connected.php`
4. Update database credentials in `connected.php`
5. Run on localhost
